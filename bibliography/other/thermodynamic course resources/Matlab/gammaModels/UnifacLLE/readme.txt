Activity coefficients by UNIFAC

See the UNIFAC grouplist examples in the gammaModels folder.

unifacCallerLLE - example of how to call the routines.
unifacSetUpLLE.m - called once to set calculate pure properties.
../unifac.m - used to calculate activities once unifacSetUpLLE.m has been run once.
unifacAijLLE - parameter database. Use 'Load unifacAijLLE.mat' to view in workspace.
