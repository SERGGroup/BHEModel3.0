Before running, use 'run ../Props/propsTableBrowse'. The propsRow
variable for each compound is in the left column. To set the
propsRow, open PreosProps.m, find "******* Input section, can be edited *****".
Set the propsRow for the desired species. Save the file.

To run the property calculator use 'run PreosPropsMenu'. 
The files PreosPropsMenu.fig and PreosProps.m will be called
from PreosPropsMenu.
